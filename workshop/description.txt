[h1]B41 Farming Interaction XP[/h1]

[b]EN[/b]

Distributes Farming XP across the full farming loop in [b]Project Zomboid Build 41[/b].

Instead of earning Agriculture progression almost entirely at harvest time, this mod rewards the main crop interactions during the full lifecycle:

[list]
[*]Plowing furrows
[*]Sowing seeds
[*]Watering crops
[*]Fertilizing crops
[*]Harvest remains vanilla
[/list]

[h1]Balance[/h1]

[list]
[*]Furrow XP: 1.0
[*]Sowing XP: 2.0
[*]Watering XP: 2.0
[*]Fertilizing XP: 4.0
[*]Harvest bonus XP: 1.0 — vanilla behavior preserved
[/list]

[h1]Installation[/h1]

Single-player:
Enable the mod in the Mods menu before loading the save.

Host multiplayer:
Enable the Workshop item and the mod in the server settings.
All players will be asked to download it before joining the server.

[b]Compatibility[/b]
[list]
[*]Targeted at Build 41.78
[*]Designed primarily for Host multiplayer
[*]Also supports single-player
[*]No dependency on other mods
[/list]
